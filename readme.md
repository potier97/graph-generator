## Scrpt para generar graficos de json o csv

Este script permite generar graficos a partir de un archivo json o csv.


## Crear un entorno virtual

```
python -m venv venv
```

## Activar el entorno virtual

MacOS y Linux
```
source venv/bin/activate
```

Windows
```
venv\Scripts\activate
```

## Instalar las dependencias

```bash
pip install -r requirements.txt
```
